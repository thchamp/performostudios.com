Performo Studios · Publiseringsguide
Live på performostudios.com
Tre faser. Ca. 20 minutter totalt. Helt gratis hosting via GitHub Pages + ditt GoDaddy-domene.
Fremgang 0 / 11 steg
Fase 1 Sett opp GitHub Pages ~8 min ▶
✓
Opprett gratis GitHub-konto
Gå til github.com og registrer deg hvis du ikke allerede har en konto. Det er helt gratis.
↗ github.com/signup
✓
Opprett et nytt repository
Klikk «New repository» (grønn knapp øverst til høyre). Gi det navnet:
performostudios.com
Viktig: Sett repository til Public. Klikk «Create repository».
✓
Last opp nettsiden din
I det nye repositoriet, klikk «uploading an existing file». Last opp filen performo-website.html som du lastet ned fra Claude.
Viktig: Gi filen nytt navn til index.html før du laster den opp. Det er dette GitHub Pages bruker som forside.
✓
Aktiver GitHub Pages
Gå til Settings → Pages (i venstre meny). Under «Source», velg «Deploy from a branch» og velg main branch. Klikk Save.
Resultat: Siden din er nå tilgjengelig på dittbrukernavn.github.io/performostudios.com — midlertidig URL mens vi kobler domenet.
✓
Legg til ditt eget domene i GitHub Pages
Fortsatt under Settings → Pages, finn feltet «Custom domain». Skriv inn:
performostudios.com
Klikk Save. GitHub vil nå vise deg noen DNS-verdier du trenger — la dette vinduet stå åpent til neste fase.
Fase 2 Koble GoDaddy til GitHub ~8 min ▶
✓
Logg inn på GoDaddy og finn DNS-innstillinger
Gå til godaddy.com → Mine Produkter → Domains → performostudios.com → DNS.
↗ Åpne GoDaddy DNS
✓
Legg til fire A-records (peker til GitHub)
Slett eksisterende A-records hvis de finnes. Legg til disse fire:
Type: A | Name: @ | Value:
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
TTL: Sett til 600 (10 minutter) for rask oppdatering.
✓
Legg til CNAME-record (for www)
Dette gjør at www.performostudios.com også fungerer:
Type: CNAME | Name: www | Value:
dittbrukernavn.github.io
(bytt «dittbrukernavn» med ditt GitHub-brukernavn)
✓
Lagre DNS-innstillingene
Klikk Save i GoDaddy. DNS-endringer tar vanligvis 5–30 minutter å spre seg — noen ganger opp til 24 timer, men sjelden.
Tålmodighet: Ikke forvent at det er live umiddelbart. Vent 15 minutter og prøv igjen.
Fase 3 Aktiver HTTPS og gå live ~5 min ▶
✓
Aktiver «Enforce HTTPS» i GitHub Pages
Gå tilbake til GitHub → Settings → Pages. Etter at DNS er oppdatert (15–30 min), vil du se en grønn hake og muligheten til å krysse av «Enforce HTTPS». Kryss av.
Hvorfor HTTPS? Nettsider uten HTTPS vises som «Ikke sikker» i nettleseren. Med HTTPS ser det profesjonelt ut og Google rangerer deg bedre.
✓
Test at alt fungerer
Åpne en ny fane og gå til:
https://performostudios.com
Fungerer det? Du skal se Performo Studios nettsiden din. Prøv også www.performostudios.com — begge skal virke.
